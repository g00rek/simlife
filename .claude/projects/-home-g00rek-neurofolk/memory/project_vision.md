Neurofolk long-term vision: evolve from rule-based simulation toward neural-network-driven civilizations.

**Why:** User wants civilizations that learn, develop skill trees, have genetics, and accumulate knowledge. Different villages/civilizations will be separate "players" with their own learning pools.

**How to apply:** Every design decision should keep the door open for:
- Per-civilization state (knowledge, learned behaviors, skill trees)
- Replacing scoring functions with trainable weights (simple NN)
- Genetic trait inheritance already exists — will expand
- Modular action system that can be driven by either rules or learned weights
